
# Дневник питания

Мобильное приложение для ежедневного учета питания, отслеживания калорий, белков, жиров и углеводов (КБЖУ), ведения истории приемов пищи и контроля рациона. 

## Функции

- Ввод и редактирование данных пользователя (имя, возраст, вес, рост, пол, уровень активности)
- Автоматический расчет суточной нормы калорий
- Добавление, редактирование и удаление приемов пищи
- Выбор даты для просмотра записей за любой день
- Отображение суммы КБЖУ за день с прогрессом относительно нормы
- Прогресс-бар визуального отображения выполнения нормы
- Создание собственных продуктов с указанием КБЖУ на 100 г
- Ведение истории всех приемов пищи
- Просмотр статистики за последние 7 дней
- Автономная работа без подключения к интернету

## Требования

- Android 8.0 (Oreo) или новее
- Оперативная память: не менее 2 ГБ
- Свободное место: не менее 100 МБ

## Установка

### Запуск готового приложения

1. Скачайте файлы проекта
2. Вставьте их в Visual Studio
3. Запустите `FoodDiary.exe`


## Использование

### Первый запуск

При первом запуске откроется экран ввода данных пользователя. Заполните все поля (имя, возраст, вес, рост, пол, уровень активности) и нажмите "Сохранить". Приложение автоматически рассчитает суточную норму калорий.

### Добавление приема пищи

1. На главном экране нажмите кнопку «+ Добавить приём пищи»
2. Выберите существующий продукт из списка или создайте новый
3. Введите вес порции в граммах
4. Нажмите «Сохранить»

### Создание нового продукта

1. При добавлении приема пищи нажмите «Создать новый продукт»
2. Введите название продукта
3. Укажите КБЖУ на 100 г (калории, белки, жиры, углеводы)
4. Нажмите «Сохранить продукт»

### Навигация по дням

- Используйте кнопки «Следующий день» и «Предыдущий день» для переключения между днями
- Нажмите на кнопку с датой для выбора конкретной даты через календарь

### Просмотр статистики

Нажмите кнопку «Статистика» на главном экране для просмотра:

- Графика изменения КБЖУ за последние 7 дней
- Средних значений за неделю
- Общей суммы калорий за каждый день
